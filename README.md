# 개요
이 저장소는 "컴퓨터 비전(기본 개념부터 최신 모바일 응용 예까지)-오일석" 책을 읽고 정리한 저장소이다.  
영상처리 및 컴퓨터비전에 대한 개념을 최대한 정리하려고 했으며 3장부터 7장이 메인이다.   
각 장은 책 정리와 과제로 나뉘며, 과제는 알고리즘 의사코드를 c++로 바꾼 코드와 결과물을 포함한다.
기본적인 openCV 함수 api를 이용한다.(ex. imread())
